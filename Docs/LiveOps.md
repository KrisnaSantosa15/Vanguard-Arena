# LiveOps Strategy - Vanguard Arena

## 1. Overview
This document defines the content cadence and retention strategies for the post-launch phase. The goal is to maintain engagement through regular updates and predictable event cycles.

## 2. Event Cadence

### 2.1 Daily Events
*   **Login Calendar**: 7-day cycle. Day 7 gives a premium reward (e.g., Hero Shard or Gem Cache).
*   **Stamina Push**: Notifications at 12:00 PM and 6:00 PM local time granting free Bonus Stamina.
*   **Daily Quests**: Simple tasks (e.g., "Play 3 Matches", "Deal 10k Damage"). Reset daily.

### 2.2 Weekly Events
*   **Double Drop Weekend**:
    *   **Time**: Friday 18:00 to Sunday 23:59.
    *   **Effect**: +100% Gold and XP from battles.
*   **Weekly Brawl**:
    *   A special game mode with rotating rules (e.g., "Cooldown Reduction 80%", "Snipers Only", "Double Speed").

### 2.3 Monthly Updates (Seasons)
*   **New Banner**: Release of 1 New Hero. High drop rate gacha banner.
*   **Side Story**: A 2-week narrative event focusing on the new hero's backstory. Rewards unique cosmetics.
*   **Balance Patch**: Buffs/Nerfs based on win-rate data from the previous month.

## 3. Battle Pass System
*   **Duration**: 30 Days (Monthly Season).
*   **Progression**: Earn "Battle Points" (BP) from Daily/Weekly quests and Match results.
*   **Levels**: 1 to 50.

### 3.1 Tracks
*   **Free Track**:
    *   Available to everyone.
    *   Rewards: Gold, Basic Upgrade Materials, Common Chests, 1 Random Emote.
*   **Premium Track (Paid)**:
    *   Cost: ~$9.99 USD.
    *   Rewards: All Free rewards + Gems, Rare Upgrade Mats, XP Boosters.
    *   **Level 50 Reward**: Exclusive Seasonal Skin (not obtainable elsewhere).

## 4. Monetization (Shop)
*   **Currency**: Gold (Soft, earned), Gems (Hard, bought).
*   **Gacha**: Hero Chests (Shards/Full Heroes).
*   **Direct Purchase**: Skins (Gem only), Daily Deals (Gold/Gem rotation).
*   **Ad Placements**:
    *   "Watch Ad to double match rewards" (Capped 3x/day).
    *   "Watch Ad for free Daily Chest".
